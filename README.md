# Recipe Finder – Web App

## Description
A recipe search web application that lets users discover delicious meals. Search by meal name or ingredient, view detailed recipes with ingredients and instructions, and save favorites to localStorage.

## Live Demo
[View Live Demo](https://maxtinka.github.io/recipe-finder)

## Features
- 🔍 Search recipes by name or ingredient
- 📸 Display recipe images
- 📝 Show ingredients with measurements
- 📖 Display cooking instructions
- ⭐ Save favorite recipes (localStorage)
- 📱 Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- TheMealDB API (free, no API key required)
- Font Awesome icons
- Google Fonts

## How to Run

### Option 1: Open Directly
1. Download or clone this repository
2. Open `index.html` in any modern web browser

### Option 2: VS Code Live Server
1. Open folder in VS Code
2. Install Live Server extension
3. Right-click `index.html` → Open with Live Server

## API Reference
This app uses [TheMealDB](https://www.themealdb.com/) – a free, open-source recipe API with no authentication required.
